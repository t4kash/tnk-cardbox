# Tenkuro Cardbox


## Setup development environment

require node.js
```bash
$ npm install -g yo grunt-cli bower generator-angular
$ gem install compass

$ git clone xxx/tnk-cardbox.git
$ cd tnk-cardbox
$ npm install
$ bower install
```

## Start local server
```bash
$ grunt serve
```

## Deploy

```bash
$ parse new parse
$ grunt build
$ cd parse
$ parse deploy
```
